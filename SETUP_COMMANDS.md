# Task Management System - Setup Commands

## ✅ Project Setup Complete!

All the code has been created for you! You just need to install the dependencies.

## Step 1: Install Backend Dependencies

Open your terminal and run:

```bash
cd server
npm install
```

## Step 2: Install Frontend Dependencies

Open a new terminal (or navigate back) and run:

```bash
cd client
npm install
```

## Step 3: Run the Application

See `HOW_TO_RUN.md` for detailed instructions on starting both the backend and frontend.

**Quick Start:**
1. Terminal 1: `cd server` → `npm run start:dev`
2. Terminal 2: `cd client` → `npm run dev`
3. Open browser: `http://localhost:5173`
